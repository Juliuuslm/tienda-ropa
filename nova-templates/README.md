# Nova Templates - Templates de Referencia

Esta carpeta contiene templates HTML/CSS de referencia que fueron utilizados como inspiración para el desarrollo de la tienda en línea Nova.

## Contenido

### ref-1: Destry Fashion E-commerce
- **Origen**: Template HTML/CSS de e-commerce de moda
- **Características**: Diseño moderno, producto detallado, carrito
- **Ruta**: `ref-1/destry/`
- **Uso**: Referencia para estructura de páginas de productos y checkout

### ref-2: Castro E-commerce Pack
- **Origen**: Template HTML/CSS e-commerce multi-propósito
- **Características**: Grid de productos, filtros, categorías
- **Ruta**: `ref-2/Castro Pack/`
- **Uso**: Referencia para diseño de listado de productos y filtros

## Propósito

Estos templates se usaron como:
- ✅ Inspiración visual y de UX/UI
- ✅ Referencia de estructura HTML
- ✅ Ideas de componentes y secciones
- ✅ Guía de mejores prácticas en e-commerce

## Notas

- Los templates originales son HTML/CSS estático
- El proyecto Nova es una conversión moderna a **Astro + React + Tailwind CSS**
- Se mantienen como referencia para futuro development y consulta

## Acceso

Abrir en navegador:
```bash
# ref-1/destry/
open ref-1/destry/index.html

# ref-2/Castro Pack/
open "ref-2/Castro Pack/index.html"
```
